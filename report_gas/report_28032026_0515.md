**Root Cause:** MT5 websocket menulis ke `gas-redis-hot` tapi overview.py membaca dari `gas-redis` (main)

**Fix di `gas-terminal-backend/src/services/redis.py`:**
```python
async def connect_hot(self):
    """Connect to hot Redis (MT5 live tick prices, no AOF)."""
    self._hot_client = redis.from_url(settings.REDIS_HOT_URL, ...)

async def get_market_price(self, symbol: str):
    await self.connect_hot()
    raw = await self._hot_client.get(f"market:{symbol.upper()}")
    return json.loads(raw) if raw else None
```

### 6. 🔐 Login Fix — ?next= URL Redirect
**Problem:** `saveSession()` di AuthContext memanggil `pushState('/')` sebelum kode bisa baca `?next=`
**Fix di `gas-terminal-frontend/src/components/LoginPage.jsx`:**
```js
// Capture ?next= BEFORE saveSession changes the URL
const next = new URLSearchParams(window.location.search).get('next');
const res = await axios.post('/auth/v1/auth/login', { username, password });
saveSession(res.data.access_token, res.data.user);
if (next) window.location.href = next;
```
