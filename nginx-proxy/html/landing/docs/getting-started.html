Getting Started with GAS

Get up and running with GAS AI Trading Terminal in under 10 minutes. This guide walks you through account creation, plan selection, terminal navigation, and your first AI analysis.

Creating Your Account

Navigate to gasstrategyai.xyz/signup and fill in the registration form. GAS supports both email/password registration and Google OAuth.

📸
Screenshot: Registration form

Required Information

  • ✓Full Name — Your display name in the platform
  • ✓Email Address — Used for login and notifications
  • ✓Password — Minimum 8 characters, mix of letters and numbers
  • ✓Country — Used for localized features and timezone
💡 Tip: Google OAuth

Sign up with your Google account for faster onboarding. No password to remember — just click "Continue with Google."

2

Email Verification

After registration, check your inbox for a verification email from noreply@gasstrategyai.xyz. Click the verification link to activate your account.

⚠️ Important

The verification link expires in 24 hours. If you don't see the email, check your spam folder or use the "Resend Verification" option on the login page.

Navigating the Terminal

The GAS terminal (app.gasstrategyai.xyz) is your main trading workspace. Here's an overview of the key sections:

📸
Screenshot: Main terminal interface

Left Sidebar — Navigation

The left sidebar contains all feature shortcuts. Click any icon to open that feature in the main panel:

  • 📊 Markets — Live price feeds for Forex, Gold, and Crypto
  • 📡 Signals — AI-generated BUY/SELL signals
  • 🔬 Technical — Multi-timeframe technical analysis
  • 📅 Calendar — Economic events calendar
  • 🛡️ Risk — Position size and risk calculator
  • ⚙️ Settings — Account settings, billing, notifications

Pair Selector

Use the pair selector at the top of the terminal to switch between trading instruments. Type the symbol name (e.g., XAUUSD, EURUSD, BTCUSDT) or browse by category.

Timeframe Buttons

Select your analysis timeframe using the buttons in the top toolbar: M1, M5, M15, M30, H1, H4, D1, W1. For swing trading, H4 and D1 are recommended. For scalping, M15 or M30.

Interpreting AI Signals

GAS AI signals contain several components. Here's how to read them:

// Example AI Signal Output
Symbol: XAUUSD (Gold)
Signal: BUY ↑
Confidence: 87.3% // Above 75% = high confidence
Timeframe: H4
Entry Zone: 2,338.00 — 2,345.00
Stop Loss: 2,324.50 // Below last swing low
Take Profit 1: 2,365.00 // 1:1.5 RR
Take Profit 2: 2,385.00 // 1:3.2 RR
Risk/Reward: 1:3.2
Validity: Next 2–3 candles

Confidence Score Guide

  • 75-100% — High confidence. Signal aligns across multiple timeframes and indicators.
  • 55-74% — Medium confidence. Consider waiting for confirmation candle.
  • Below 55% — Low confidence. Skip this signal or use smaller position size.
7

Setting Up Risk Manager

The AI Risk Manager (Plus plan and above) automatically calculates optimal position sizes based on your account balance and risk tolerance. Configure it once for consistent money management:

Account Balance

Enter your current trading account balance (e.g., $1,000 USD). This is used for position size calculation.

Risk Per Trade

Recommended: 1-2% per trade. GAS defaults to 1%. With $1,000 balance, max risk = $10 per trade.

Max Daily Loss Limit

GAS will warn you when you approach your daily loss limit. Recommended: 5% of account balance.

💡 Pro Tip: Prop Firm Mode

If you're trading a prop firm account (FTMO, MyForexFunds, etc.), enable "Prop Firm Mode" in Risk Manager. GAS will automatically enforce the firm's drawdown rules and alert you before violations.

API Usage Examples

The GAS API allows you to programmatically access all platform features. Here are quick examples to get you started:

Authentication

# Python: Get API token
import requests

response = requests.post(
    "https://app.gasstrategyai.xyz/auth/v1/login",
    json={
        "email": "your@email.com",
        "password": "your_password"
    }
)
token = response.json()["access_token"]
headers = {"Authorization": f"Bearer {token}"}

Get AI Signal via cURL

curl -X POST https://app.gasstrategyai.xyz/web/api/v1/analysis/signal \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"symbol": "EURUSD", "timeframe": "H1"}'

JavaScript Fetch Example

const signal = await fetch(
  "https://app.gasstrategyai.xyz/web/api/v1/analysis/signal",
  {
    method: "POST",
    headers: {
      "Authorization": `Bearer ${token}`,
      "Content-Type": "application/json"
    },
    body: JSON.stringify({
      symbol: "XAUUSD",
      timeframe: "H4"
    })
  }
);
const data = await signal.json();
console.log(data.signal, data.confidence);
